## ✊🏼🖐🏼✌🏼 Jokenpô

## 📖 Sobre 

O Jokenpô é um projeto desenvolvido para simular o clássico jogo de Pedra, Papel e Tesoura de forma interativa e dinâmica. A aplicação permite que o usuário jogue contra o computador, que realiza suas escolhas de maneira aleatória a cada rodada.

O projeto foi criado com foco na prática de JavaScript, explorando conceitos como manipulação do DOM, eventos, funções, condicionais, geração de números aleatórios e atualização dinâmica da interface. Além disso, a interface foi desenvolvida para ser simples, intuitiva e responsiva, proporcionando uma boa experiência em diferentes dispositivos.

## ✨ Funcionalidades

- 🎮 Jogue contra o computador.
- 🤖 Escolha aleatória do computador a cada rodada.
- 🏆 Verificação automática do vencedor.
- 📊 Atualização do placar em tempo real.
- 🔄 Possibilidade de jogar quantas vezes desejar.
- 📱 Interface responsiva para diferentes tamanhos de tela.
- 🎨 Design moderno e intuitivo.

## 🛠️ Tecnologias Utilizadas

- HTML5 🟧
- CSS3 🟦
- JavaScript 🟨

## 💡 Conceitos Praticados

Durante o desenvolvimento deste projeto foram aplicados diversos conceitos importantes do Front-End, como:

- Manipulação de elementos com JavaScript.
- Eventos de clique.
- Estruturas condicionais (if, else).
- Funções.
- Arrays.
- Geração de números aleatórios (Math.random()).
- Atualização dinâmica do conteúdo da página.
- Organização do código.
- Responsividade utilizando CSS.

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de fortalecer os conhecimentos em HTML, CSS e JavaScript, colocando em prática conceitos fundamentais da programação Front-End através de uma aplicação interativa e divertida.
